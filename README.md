## Setting up seashell
Clone the project to retrieve the sources:
```
$>git clone https://github.com/sebastienivanez/poker.git
```

Build the project using CMake:
```
$>cmake . -Bbuild
```

Finally, compile the sources and generate the binary:
```
$>cmake --build build
```
